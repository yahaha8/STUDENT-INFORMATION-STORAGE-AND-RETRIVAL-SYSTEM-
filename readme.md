##Project Title-STUDENT INFORMATION STORAGE AND RETRIVAL SYSTEM (Python Beginner Project)

###Overview of the Project
This project is a simple Python-based student information system where I can add, store, and search student details.
The data gets saved In a text file (storeddata.txt), and whenever needed, the program can fetch and display the information.
I made this project as a beginner, so the code is very simple and easy to follow.



1##Features
a]Add new student details (name, branch, registration number, city, etc.)
b]Store all details safely in a text file
c]Search student details by name
d]Keep adding multiple entries in one go
e]Beginner-level menu-driven interface
f]Prevents invalid input (basic error handling)


2##Technologies / Tools Used
a]Python
b]File handling using .txt
c]Basic loops, conditions, and dictionaries


3##Steps to Install & Run the Project
1.	Download/cl0ne the project folder.
2.	Make sure Python is installed on your system.
3.  Keep the Python file and storeddata.txt in the same folder.
4.	Open a terminal / command prompt.
5.	Run the file using:
         Python main.py
6.	Follow the menu on screen to add or search students.
           



3##Instructions for Testing
a]Choose option 1 to search for a student by name.
b]Choose option 2 to add new student details.
c]Check storeddata.txt to see whether data is being stored correctly.
d]Try adding multiple students and then searching for them.
