###PROBLEM STATEMENT
Managing student information manually becomes confusing because the details are scattered and not organized properly.
There is no simple way to store the data or check it later without searching through multiple pages.
So, I wanted to make a very basic system where I can add student details and also search them whenever needed.


###SCOPE OF THE PROJECT
a]The project focuses on creating a small menu-based Python program that allows the user to:
b]Add student data
c]Store it safely in a text file
d]Fetch and display the stored data when required
